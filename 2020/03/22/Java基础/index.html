第一章 JAVA语言概述

一、JAVA程序的种类

  1. Java小应用程序（Java Applet）

    • 在Web浏览器中运行（内嵌Java虚拟机）

    • 特性标记

  2. Java应用程序（Java APPlication）

二、第一个Java程序

2.1 代码编写

  • Java源程序名与程序内部的类名称必须一致

    所保存的文件名为：MyClass.java

    注意：大小写敏感

2.2如何运行程序？

  1. 需要先使用cmd命令行，输入javac Helloworld.java进行编译；
  2. 编译完成后当前文件夹会创建Helloworld.class文件；
  3. 通过java Helloworld运行程序。

第二章 JAVA语法基础

一、语法规则

1.1程序设计语言中的5类符号

  • 关键字（保留字）
  • 标识符
  • 分隔符
  • 常量数据
  • 运算符

二、数据类型

2.1基本数据类型

  • 数字（number）
    • 整型（integers）
      1. 字节整数（byte, 8 bits）
      2. 短整数（short, 16 bits）
      3. 整数（int, 32 bits）
      4. 长整数（long, 64 bits）
    • 实型（real numbers）：浮点数
      1. 单精度（float, 32 bits）
      2. 双精度（double, 64 bits）
  • 字符（char, 16 bits Unicode 字符）
  • 布尔（boolean）

2.2 复合数据类型

  • 数组（Array），类（class），接口（interface）

2.3 常量数据表示

  • 示例

2.4 类型转换（casting）

三、常量和变量

3.1常量

  • 程序执行过程中，值保持不变的量
  • 整型常量
    • 常用十进制、八进制、十六进制表示
    • 有正负号
  • 实型常量
    • 双精度实数（double，8个字节，数字后加字母D或d）
    • 浮点实数（float，4个字节，数字后加字符F或f）
    • 若无明确字母表示，则系统默认为双精度实数
  • 布尔型常量
    • true
    • false
  • 字符型常量
    • 用单引号括起来的单个字符
    • JAVA中的字符为Unicode字符
  • 字符串常量

3.2变量

  • 程序执行过程中，值可以改变的量

  • 整型变量、实型变量、字符型变量、字符串变量、布尔变量等

  • 变量定义

  • 变量初始值/初始化

  • 在变量声明时赋值

  • 采用赋值语句

  • 变量的作用域

  • final变量

    • 带有关键字final的变量
    • final变量初始化后不能再改变

四、运算符和表达式

4.1 运算符

  1. 算术运算符
  2. 关系运算符
    • 比较运算，计算结果true或flase
  3. 逻辑运算符
  4. 位运算符
  5. 移位运算符
  6. 条件运算符
